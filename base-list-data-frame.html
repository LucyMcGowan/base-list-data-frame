Lists

An R list is a recursive vector. It’s a data structure consisting of a collection of components. The components can be of different types (ie character, numeric) disguised as model objects, data frames, list-columns within data frames, and more.

For example, a list could be a collection of different types of vectors of different lengths, as seen below.

list(a = 1:3, b = "a string", c = pi, d = list(-1, -5))
## $a
## [1] 1 2 3
## 
## $b
## [1] "a string"
## 
## $c
## [1] 3.141593
## 
## $d
## $d[[1]]
## [1] -1
## 
## $d[[2]]
## [1] -5

Components of the list are numbered. These components are also known as sub-litss and can be extracted using a double bracket. For example, you can reference the first element of a list with [[1]] as seen below.

x <- list(a = 1:3, b = "a string", c = pi, d = list(-1, -5))
x[[1]]
## [1] 1 2 3

Components can also be referenced via their names. For example, the first element of the list below could be referenced by [["a"]]

x[["a"]]
## [1] 1 2 3

Data frames

A data frame is a specific type of list with the class data.frame. Tibbles are a specific subclass of data.frame and one of the unifying features of the tidyverse. Data frames consist of rows and columns.

For example, here is a data frame with three columns, x, y, and z each with four rows.

data.frame(
  x = c(2, 4, 6, 8),
  y = c(10, 12, 14, 16),
  z = c(18, 20, 22, 24)
)

For example, here is a tibble data frame with three columns, x, y, and z.

tibble(
  x = (1:5), 
  y = (1), 
  z = (x^ 2 + y)
)
